About nphysics

nphysics is a 2 and 3-dimensional physics engine for games and animations.

Its most distinctive feature is its genericity wrt the simulation dimension.Â That means you can use it for both 2-dimensional physics and 3-dimensional physics. Higher dimensions could be possible, but nphysics has not be written/tested with those in thought.

Features

  • static and dynamic rigid bodies.
  • common convex primitives: cone, box, ball, cylinder.
  • concave geometries build from convex primitives (aka. compound geometries).
  • stable stacking.
  • island based sleeping (objects deactivation).
  • ray casting.
  • swept sphere based continuous collision detection.
  • ball-in-socket joint.
  • fixed joint.

For more, check out the documentations of the different variants of the library:

  • nphysics2df32, for a 2d physics engine using single-precision numbers.
  • nphysics3df32, for a 3d physics engine using single-precision numbers.
  • nphysics2df64, for a 2d physics engine using double-precision numbers.
  • nphysics3df64, for a 3d physics engine using double-precision numbers.
